## 2. Status Keys

| Status | Meaning |
| --- | --- |
| `planned` | Target identified, no message sent yet |
| `drafted` | Message drafted and ready |
| `sent` | Initial outreach sent |
| `follow-up due` | Waiting on reply and follow-up date reached |
| `responded` | Reply received, decision pending |
| `approved` | Clear permission or workable arrangement received |
| `declined` | Clear no or incompatible terms |
| `parked` | Deliberately paused |

## 4. Operating Rule

Whenever an email is sent:

1. change `Status` to `sent`;
2. fill `Date` with the actual send date if different from draft date;
3. set a `Next follow-up date`;
4. summarise any reply in `Outcome / notes`;
5. update the source register if permission status changes.
