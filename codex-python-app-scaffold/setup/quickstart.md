# Quickstart

## Goal

Adapt this scaffold to a Python project in less than 15 minutes.

## Steps

1. Copy the scaffold files into the target repository.
2. Fill out `setup/questionnaire.md`.
3. Replace placeholders in `CLAUDE.md` and `CONTEXT.md`.
4. Update `references/python-routing.md` with the repo's actual entry points and preferred test targets.
5. Update `_config/change-safety.md` with project-specific import, environment, and path constraints.
6. Add or update `.gitignore` rules for `.env`, local databases, caches, generated outputs, and temp files.
7. Sanity-check the repo by reading the routing files as if you were a fresh Codex session.

## Minimum Required Customization

If time is tight, only customize:

- `CLAUDE.md`
- `CONTEXT.md`
- `references/python-routing.md`
- `_config/change-safety.md`
