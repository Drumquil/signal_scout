# Stage 05: Handoff

## Goal

Close the loop with a concise summary and capture improvements to the source of
truth when patterns emerge.

## Inputs

- Layer 3: `_config/workspace-principles.md`
- Layer 4: changed files
- Layer 4: validation notes from stage 04

## Process

- Summarize what changed and what was verified.
- Note residual risks or skipped checks.
- If the same kind of fix is likely to recur, identify the source file or
  instruction that should be improved next.

## Outputs

- Handoff notes in `output/`
- Source-improvement follow-ups when warranted
