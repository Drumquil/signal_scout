# Quickstart

## Goal

Adapt this scaffold to a new project in less than 15 minutes.

## Steps

1. Copy the scaffold files into the target repository.
2. Fill out `setup/questionnaire.md`.
3. Replace placeholders in `CLAUDE.md` and `CONTEXT.md`.
4. Update `_config/change-safety.md` with the project's path and validation constraints.
5. Adjust any stage `CONTEXT.md` files for domain-specific routing.
6. Update `.gitignore` if the project needs to ignore scaffold outputs or sensitive inputs.
7. Sanity-check the repo by reading the routing files as if you were a fresh Codex session.

## Minimum Required Customization

If time is tight, only customize:

- `CLAUDE.md`
- `CONTEXT.md`
- `_config/change-safety.md`

The rest can remain generic until the project shows a need for tighter routing.
